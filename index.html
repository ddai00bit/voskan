How do i get started?

We suggest you watch the getting started tutorial, which can be found in the main directory. After that watch the other basic tutorials on voskan creator. Once you know all about the basic stuff (which is really simple) you realise voskan revolves around one simple idea: Select blocks you like from creator, Then create your own layout using the blocks code in the starter pack. Voskan provides basic customization but if you like you can also dig in deeper into the coding sutff for further customization.
