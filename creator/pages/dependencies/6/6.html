Dependencies > Stellar



Stellar is a javascript library used for creating parallax backgrounds and elements. It is used by block 11 to create parallax background effect. You can find more about it at its official website

A note on parallax backgrounds

If you use a high quality image as a parallax background, it might result in a sluggish and laggy scroll. To deal with this problem you have to reduce your image size by about 50% (or what suit your needs) using Simple Image Resizer
