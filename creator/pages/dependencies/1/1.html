Dependencies > Browser/Window frame




Description

The browser frame or the window frame is a container for you to elegantly preview your product in.

It makes your customers feel like they are 'really' using the product first hand. Nevertheless, it also adds style.

Below you can find the code for the browser/window and some customization classes to further customize this dependency.
