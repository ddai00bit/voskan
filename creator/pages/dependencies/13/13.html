Dependencies > Animate.css



Animate.css is a very popular CSS animation library. It has thousands of CSS animations for you to use. You can check the list of animations at its official website

Usage

Say i want to add a animation to this heading element

1
<h1>Lorem animated</h1>

I would select my desired animation from the animation list and then add it to the element using the class attribute.

1
<h1 class="animated fadeInRight">Lorem animated</h1>
